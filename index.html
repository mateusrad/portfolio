FOLHA Nº 02 — SOBRE

Antes do código, o problema real

Minha trajetória começou na área da saúde, atuando como técnico em radiologia. Aprendi a lidar diariamente com processos críticos, informações sensíveis e protocolos rígidos que exigem exatidão absoluta, sem espaço para erros. Hoje, também formado em Tecnologia em Análise e Desenvolvimento de Sistemas (TADS — UDESC), aplico essa mesma exigência e rigor na construção de software: mapeio como o trabalho realmente acontece na sua operação e desenho um sistema que se encaixa perfeitamente nesse fluxo, não o contrário.

Trabalho principalmente com Python e Django no backend, integrações com Firebase quando o projeto exige dados em tempo real, e um front-end direto ao ponto. Cada projeto abaixo não é apenas código, mas uma solução nascida de uma necessidade concreta.

  • Python
  • Django
  • Firebase
  • JavaScript
  • PostgreSQL
  • Git / GitHub
